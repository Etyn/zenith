# Cartes agent Zenith — Jupiter (vert / teal)

> Feuille de validation assemblée à partir des brouillons de décodage
> (`.superpowers/sdd/cards-decode-*.md`). **Rien n'est inventé** : ce document réorganise fidèlement
> le contenu des brouillons, il ne réinterprète pas les icônes.

**Couleur → planète** : violet = Mercure · orange = Vénus · bleu = Terra · rouge = Mars · vert/teal = **Jupiter**.

**Règle constante** : le 1er effet d'une carte est toujours `+1 influence sur la planète de la carte`.

**Légende confiance** : ✓ = lecture confiante (conforme au lexique confirmé) · ❓ = incertaine (hypothèse posée, à confirmer) · ❓❓ = très incertaine / icône non mappée (description littérale gardée).

**Légende type** : **OBLIGATOIRE** = icônes sans flèche de liaison (toujours appliqué) · **FACULTATIF** = action reliée par une flèche/chevron ▶ (y compris les conditions « ! » et les échelles de paliers) — le joueur choisit de l'activer ou non. Quand le brouillon lui-même ne tranche pas (ex. séparateur « / » non documenté), le Type est marqué ❓.

**Colonne « Réel »** : à remplir/corriger par l'utilisateur après vérification sur les cartes physiques.

13 cartes dans cette feuille (sur 90 au total, toutes planètes confondues).

> ⚠️ **Note générale sur la couleur** : le vert Jupiter peut sembler « bleu-teal » selon l'éclairage du scan
> (cf. guide de décodage), ce qui rend la couleur de bandeau incertaine sur la quasi-totalité des cartes de
> cette feuille (marquée ❓ sur chacune) — à confirmer visuellement carte par carte avec l'utilisateur.
